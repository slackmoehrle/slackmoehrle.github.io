Creating a scene

Creating a scene

Before we can do anything we need a Scene object to hold everything. Our first sene will take care of introducing the player to out game and allowing them to start the game by pressing play. As previously mentioned this Scene will have a background, text music and a play button so the player can start the game.

To create a Scene:

  • Right click on Scene in the Assets panel and select Create and then Scene.

  • Name the Scene to MainScene (notice no spaces!)

  • Double click on the MainScene to open it.

  • In the Properties panel for the Scene make sure to select the checkboxes for Auto Release Assets and Async Load Assets.

With these steps done, we can now start working with our new Scene. Double click on the MainScene and lets move on to laying out our user interface!

Task: Now is a good time to save your project! From the File menu, select Save Scene or use your operating systems shortcut key.
